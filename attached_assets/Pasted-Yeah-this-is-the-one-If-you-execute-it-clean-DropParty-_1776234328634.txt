# ⚙️ How It Works (MVP flow)

### 👤 Creator side

1. Enter:

   * Total amount (e.g. $100)
   * Number of slots (e.g. 50)
2. Contract calculates:

   * $2 per claim
3. Click **Create Drop**
4. Get link:

   ```
   yourapp.xyz/drop/0xabc123
   ```

---

### 👥 User side

1. Open link
2. Connect wallet
3. Click **“Claim”**
4. Instantly receive USDC

👉 If all slots filled → show:

> “Drop finished ⚡”

---

# 🧩 Smart Contract (super simple logic)

You only need:

### State:

* `totalAmount`
* `amountPerUser`
* `maxClaims`
* `claimedCount`
* `mapping(address => bool) claimed`

---

### Functions:

#### 1. `createDrop()`

* Store drop details
* Transfer USDC from creator

---

#### 2. `claim()`

* Check:

  * not already claimed
  * claims < max
* Send USDC
* mark claimed

---

#### 3. `refundRemaining()`

* After expiry (optional)

---

👉 That’s it. No complex logic needed.
